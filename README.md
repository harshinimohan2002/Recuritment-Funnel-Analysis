# Recruitment Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes the recruitment process using key hiring metrics and interactive visualizations.

## Tools Used
- Power BI
- DAX

## KPIs
- Total Applications
- Total Hired
- Hiring Rate
- Applications by Job Title
- Applications by Source

## Dashboard Features
- Interactive slicers
- Recruitment KPIs
- Hiring analysis
- Candidate source analysis

## Key Insights
- Tracks hiring performance.
- Identifies the most effective recruitment sources.
- Naukri was one of the recruitment sources contributing to candidate applications in the selected data.
